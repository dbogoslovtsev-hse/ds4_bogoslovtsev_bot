БОТ предоставляет информацию по компаниям, зарегистрированным на habr.com.
Доступна общую статистику по странам, городам, хабам. Например, количество
компаний в стране.

Мможно посмотреть ТОП-20 компаний по рейтингу, количеству публикаций или подписчиков.

Пользователь может запросить информацию по копании, или построить рейтинг ТОП-20 по одному из
возможных полей в таблице.

Список команд:

reset - сброс текущего выбора параметров
start - начинаем работу заново
info - подробная информация о моих возможностях
commands - перечисление полного списка команд
listcities - перечисление списка ТОП-20 городов
listcountries - перечисление списка ТОП-20 стран
listhabs - перечисление списка ТОП-20 хабов
listfields - перечисление списка полей
topraiting - список ТОП-20 компаний по рейтингу
topposts - список ТОП-20 компаний по количеству публикаций
topsubscribers - список ТОП-20 компаний по количеству подписчиков

В качестве источника данных используется зарнее собранный список компаний с сайта habr.com в рамках ДЗ
по Beautiful Soup и Selenium.
